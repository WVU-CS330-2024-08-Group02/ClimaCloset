# The `Shoes` Folder

The `Shoes` folder contains the jsx and css that create, format, and modify Shoes. It displays the shoes submission form where the user can input the shoes they own. Upon submission of the form, the user's response will be stored in the closet.

## `Shoes` Folder/File Structure and Overview

### Structure

```
frontend/
├── src/      # Modifies the UI design and handles user interactions
│   ├── components/    # Non-page components of the app
│   │   ├── Shoes/          # Creates and formats the Shoes box
│   │   │   ├── Shoes.css          # Controls appearance and layout of Shoes
│   │   │   ├── Shoes.jsx          # Sets up and implements the Shoes box
│   │   │   └── README.md
```

### Overview

1. **`Shoes.css`**
   - This file controls the appearance and layout of Shoes. It handles styling all of the interactive and responsive elements.

2. **`Shoes.jsx`**
   - This file contains all of the elements and implementation for Shoes. It displays the submission form as a checkbox list with the shoes listed, and it handles the submission of the form to the closet storage.
