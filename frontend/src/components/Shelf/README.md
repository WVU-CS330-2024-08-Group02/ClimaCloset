# The `Shelf` Folder

The `Shelf` folder contains the jsx and css that create, format, and modify Shelf. It creates the closet with the four different shelves for tops, bottoms, shoes, and accessories, and it displays the clothing icons within their respective shelf. 

## `Shelf` Folder/File Structure and Overview

### Structure

```
frontend/
├── src/      # Modifies the UI design and handles user interactions
│   ├── components/    # Non-page components of the app
│   │   ├── Shelf/         # Creates and formats the Shelf
│   │   │   ├── Shelf.css         # Controls appearance and layout of Shelf
│   │   │   ├── Shelf.jsx         # Sets up and implements the Shelf
│   │   │   └── README.md
```

### Overview

1. **`Shelf.css`**
   - This file controls the appearance and layout of Shelf. It handles styling all of the interactive and responsive elements.

2. **`Shelf.jsx`**
   - This file contains all of the elements and implementation for Shelf. It creates the closet where user clothing is stored. Within the closet, four shelves are created for tops, bottoms, shoes, and accessories. As the user update's their clothing, the respective clothing icons are displayed in their respective shelf.
