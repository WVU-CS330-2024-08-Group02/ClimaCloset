# The `Accessories` Folder

The `Accessories` folder contains the jsx and css that create, format, and modify Accessories. It displays the accessories submission form where the user can input the accessories they own. Upon submission of the form, the user's response will be stored in the closet. 


## `Accessories` Folder/File Structure and Overview

### Structure

```
frontend/
├── src/      # Modifies the UI design and handles user interactions
│   ├── components/    # Non-page components of the app
│   │   ├── Accessories/       # Creates and formats the Accessories box
│   │   │   ├── Accessories.css          # Controls appearance and layout of Accessories
│   │   │   ├── Accessories.jsx          # Sets up and implements the Accessories box
│   │   │   └── README.md
```

### Overview

1. **`Accessories.css`**
   - This file controls the appearance and layout of Accessories. It handles styling all of the interactive and responsive elements.

2. **`Accessories.jsx`**
   - This file contains all of the elements and implementation for Accessories. It displays the submission form as a checkbox list with the accessories listed, and it handles the submission of the form to the closet storage.
