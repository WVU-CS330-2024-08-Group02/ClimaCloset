# The `Tops` Folder

The `Tops` folder contains the jsx and css that create, format, and modify Tops. It displays the tops submission form where the user can input the tops they own. Upon submission of the form, the user's response will be stored in the closet.

## `Tops` Folder/File Structure and Overview

### Structure

```
frontend/
├── src/      # Modifies the UI design and handles user interactions
│   ├── components/    # Non-page components of the app
│   │   ├── Tops/         # Creates and formats the Tops box
│   │   │   ├── Tops.css        # Controls appearance and layout of Tops
│   │   │   ├── Tops.jsx        # Sets up and implements the Tops box
│   │   │   └── README.md
```

### Overview

1. **`Tops.css`**
   - This file controls the appearance and layout of Tops. It handles styling all of the interactive and responsive elements.

2. **`Tops.jsx`**
   - This file contains all of the elements and implementation for Tops. It displays the submission form as a checkbox list with the tops listed, and it handles the submission of the form to the closet storage.
