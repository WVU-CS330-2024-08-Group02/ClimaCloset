# The `CenterContainer` Folder

The `CenterContainer` folder contains the jsx and css that create, format, and modify CenterContainer. It creates a wrapper component the provides a centered container on the app for other componenets. 

## `CenterContainer` Folder/File Structure and Overview

### Structure

```
frontend/
├── src/      # Modifies the UI design and handles user interactions
│   ├── components/    # Non-page components of the app
│   │   ├── CenterContainer/       # Creates and formats the CenterContainer
│   │   │   ├── CenterContainer.css          # Controls appearance and layout of CenterContainer
│   │   │   ├── CenterContainer.jsx          # Sets up and implements the CenterContainer
│   │   │   └── README.md
```

### Overview

1. **`CenterContainer.css`**
   - This file controls the appearance and layout of CenterContainer. It handles styling all of the interactive and responsive elements.

2. **`CenterContainer.jsx`**
   - This file contains all of the elements and implementation for CenterContainer. It creates the wrapper component that provides structure to other components in the app.
