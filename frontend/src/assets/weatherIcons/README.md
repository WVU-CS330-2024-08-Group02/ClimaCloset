# The `weatherIcons` Folder

The `weatherIcons` folder contains all of the weather icons displayed with the hourly and daily forecast on the Weather page. These icons get updated as the day progresses.  

## `weatherIcons` Folder/File Structure and Overview

```
frontend/
├── src/        # Includes the entire app set-up and UI design, and handles any user interactions
│   ├── assets      # Holds all images for the app
│   │   ├── weatherIcons    # Weather icons for weather forecast
│   │   │   ├── Cloud.png
│   │   │   ├── Night.png     
│   │   │   ├── nighttime.png
│   │   │   ├── PlaceholderRainy.png
│   │   │   ├── Rain.png
│   │   │   ├── README.md
│   │   │   ├── Snowy.png
│   │   │   ├── Stormy.png
│   │   │   └── Sun.png
```
