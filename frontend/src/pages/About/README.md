# The `About` Folder

The `About` folder contains the jsx and css that create, format, and modify the About page. This page displays a brief overview of ClimaCloset as a whole, and then show pictures and names and positions of the ClimaCloset team members. 

## `About` Folder/File Structure and Overview

### Structure

```
frontend/
├── src/      # Modifies the UI design and handles user interactions
│   ├── pages/    # All pages in the app
│   │   ├── About/       # Creates and formats About page
│   │   │   ├── About.css          # Controls appearance and layout of About page
│   │   │   ├── About.jsx          # Adds all elements of the About page
│   │   │   └── README.md
```

### Overview

1. **`About.css`**
   - This file controls the appearance and layout of the About page. It handles styling all of the interactive and responsive elements of the page that users will come into contact with.

2. **`About.jsx`**
   - This file contains all of the elements on the About page. These elements are the team member name, team member picture 1 & 2, team member position, and the overview of ClimaCloset as a whole.
