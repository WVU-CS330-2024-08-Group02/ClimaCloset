# The `Profile` Folder

The `Profile` folder contains the jsx and css that create, format, and modify the Profile page. This page displays the user's name and profile picture, and it allows the user to edit their profile information. 

## `Profile` Folder/File Structure and Overview

### Structure

```
frontend/
├── src/      # Modifies the UI design and handles user interactions
│   ├── pages/    # All pages in the app
│   │   ├── Profile/       # Creates and formats Profile page
│   │   │   ├── Profile.css          # Controls appearance and layout of Profile page
│   │   │   ├── Profile.jsx          # Adds all elements of the Profile page
│   │   │   └── README.md
```

### Overview

1. **`Profile.css`**
   - This file controls the appearance and layout of the Profile page. It handles styling all of the interactive and responsive elements of the page that users will come into contact with.

2. **`Profile.jsx`**
   - This file contains all of the elements on the Profile page. It displays the user's name and profile picture, and it allows them to edit username, change password, update email, change profile picture, or log out of the app.
