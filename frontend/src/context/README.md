# The `context` Folder

The `context` folder contains the `AuthContext.jsx` file, which provides authentication state management through React context. This allows for other components of the app to access login status, user data, and authentication functions. 

## `context` Folder/File Structure and Overview

```
frontend/
├── src/      # Modifies the UI design and handles user interactions
│   ├── context/    # Holds authentication state management
│   │   ├── AuthContext.jsx     # Creates authentication state management
│   │   └── README.md
```
