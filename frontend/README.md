# The `frontend` Folder

The `frontend` folder contains the user-side components of the project (a React-based frontend). This will include setting up the design of the app, displaying information from the `backend`, and any interactions the user has with the app. 

## `frontend` Folder/File Structure and Overview

### Structure

```
frontend/
├── public/     # Contains files intended for public access
├── src/        # Modifies the UI design and handles user interactions
└── README.md
```

### Overview

1. **`public`**
   - This folder contains files intended for public access. .

2. **`src`**
   - This folder contains all of the code that creates the UI design, user interactions, and API requests for the webapp.
