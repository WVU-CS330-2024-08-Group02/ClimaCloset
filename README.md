# ClimaCloset

ClimaCloset is a web-based application that is designed to help you pick the ideal outfit based on the current weather in your area. 
By accessing our weather and climate data from the National Oceanic and Atmospheric Administration (NOAA)'s Planetary Computer from the Global Historical Climatology Network daily (GHCNd), ClimaCloset generates and suggests outfits from the user's own customized wardrobe. Users have the ability to manange/view their wardrobe, add new outifts, and view the weather conditions in their local area.

## Features

- **Weather-Based Outfit Suggestions:** ClimaCloset recommends outfits based on the current climate data to make sure you are always dressed appropiately.
- **Customized Closet:** Input and view your own personalized collection of outfits.
- **Add a New Outfit(s):** Easily add outfits to your virtual closet.
- **Weather View:** Access current weather information straight from the app to help plan your day.

## How It Works
1. **Connect to NOAA Data**: ClimaCloset gains access to its weather data using NOAA's API to get current information (temperature, precipitation, wind conditions, etc).
2. **Outfit Generation/Selection**: From the current weather that was viewed, the application will suggest the most appropiate outift from your closet.
3. **Wardrobe Management**: Users can add new outfits, categorize them by clothing type (shirt, pants/shorts, shoes, accessories, etc.), and view them all in their closet.
